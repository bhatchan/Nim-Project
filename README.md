# Nim Project

## What is Nim?

Nim is an impartial, combinatorial game where players take turns removing (or "nimming") sticks from a pile or piles of sticks. In a game of traditional Nim played with `n` sticks, players can remove anywhere from 1 to `n` sticks on their turn, and the player to remove the last stick wins (conversely, the player that cannot remove any sticks loses).

For this project, `Nim(1, x, y)` describes a game where players are allowed to remove exactly 1, `x`, or `y` sticks on their turn. Given such a game `Nim(1, x, y)` with `n` sticks, one of the two players will always win if they play perfectly (positional advantage). 

## Module 1

In module 1, programs were created to investigate win patterns for differing `Nim(1, x, y)` games as `n` increases.

## Module 2

In module 2, programs were created to simulate 1-pile nim games between imperfect players.
